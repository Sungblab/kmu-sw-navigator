# RAG 원천자료 접수 템플릿

사용자가 PDF, 사진, 캡처, 엑셀, 텍스트를 줄 때 아래 항목을 함께 채운다. 원본 파일명과 출처를 분리해 두면, 나중에 `data/raw/*.md`와 Supabase `document_chunks`의 근거 표시를 정확히 맞출 수 있다.

## 기본 정보

- 자료 제목:
- 원본 파일명 또는 URL:
- 공식 출처명:
- 수집일:
- 담당자가 확인한 날짜:
- 적용 대상: 소프트웨어학부 / 인공지능학부 / 소프트웨어융합대학 전체 / 기타
- 대상 학년: 1학년 / 2학년 / 3학년 / 4학년 / 전체
- RAG category: `curriculum` / `track` / `roadmap` / `club` / `system` / `career` / `startup` / `freshman` / `general`
