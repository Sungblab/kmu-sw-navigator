# 소프트웨어융합대학 동아리와 활동 안내

신입생은 관심 분야에 따라 개발 동아리, 알고리즘 스터디, 보안 스터디, 인공지능 프로젝트 팀, 창업 활동을 탐색할 수 있다.

## 개발과 서비스 제작

웹 서비스 개발에 관심이 있다면 React, FastAPI, 데이터베이스를 다루는 프로젝트형 동아리나 팀 프로젝트가 잘 맞는다.

## 알고리즘과 코딩 테스트

알고리즘 문제 해결에 관심이 있다면 주간 문제 풀이 스터디, ICPC 준비 모임, 코딩 테스트 대비 스터디를 추천한다.

## 인공지능과 데이터

AI 모델, 데이터 분석, 추천 시스템에 관심이 있다면 Python 기반 프로젝트, Kaggle 스터디, 논문 읽기 모임을 탐색한다.

## 보안

정보보호에 관심이 있다면 웹 해킹, 네트워크, 암호 기초를 함께 공부하는 스터디가 도움이 된다.
