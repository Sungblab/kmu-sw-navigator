# 신입생 관심 분야별 학습 로드맵

로드맵은 정답이 아니라 첫 학기 탐색을 돕는 기준이다. 학생은 관심 분야, 선수 지식, 프로젝트 경험에 따라 순서를 조정할 수 있다.

## AI와 데이터 분석

Python 문법, 기초 수학, 자료구조, 통계, 머신러닝 기초, 데이터 분석 프로젝트 순서로 학습한다. 작은 데이터셋으로 분석 결과를 설명하는 경험이 중요하다.

## 웹과 앱 서비스

HTML, CSS, JavaScript, React, API, 데이터베이스, 배포 흐름을 학습한다. 신입생에게는 로그인 없는 간단한 웹앱을 먼저 완성해 보는 것이 좋다.

## 정보보호

네트워크 기초, 운영체제, 웹 구조, 인증/인가, 암호 기초, 모의해킹 윤리 순서로 학습한다. 실습은 반드시 허가된 환경에서 진행한다.

## 게임과 인터랙티브 콘텐츠

Python 기초, 이벤트 루프, 간단한 물리/충돌 처리, Unity 또는 웹 canvas 기초를 학습한다. 작은 게임을 끝까지 완성하는 경험이 중요하다.
