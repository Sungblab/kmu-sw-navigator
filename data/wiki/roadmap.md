# 신입생 관심 분야별 학습 로드맵

로드맵은 정답이 아니라 첫 학기 탐색을 돕는 기준이다. 학생은 관심 분야, 선수 지식, 프로젝트 경험에 따라 순서를 조정하고, 실제 수강은 적용 교과과정과 개설 과목을 확인한 뒤 결정한다.

## AI와 데이터 분석

Python 문법, 기초 수학, 자료구조, 통계, 머신러닝 기초, 데이터 분석 프로젝트 순서로 학습한다. 인공지능학부 교과과정의 Basic/Advance 단계와 소프트웨어학부 빅데이터/머신러닝 트랙 방향을 함께 참고한다.

## 웹과 앱 서비스

HTML, CSS, JavaScript, React, API, 데이터베이스, 배포 흐름을 학습한다. 소프트웨어학부 웹/정보보호, 엔터프라이즈SW 방향과 연결해 서비스 구현, 보안, 아키텍처를 단계적으로 비교한다.

## 정보보호

네트워크 기초, 운영체제, 웹 구조, 인증/인가, 암호 기초, 모의해킹 윤리 순서로 학습한다. 실습은 반드시 허가된 환경에서 진행하고, 공식 트랙의 웹/정보보호 과목 흐름과 함께 확인한다.

## 게임과 인터랙티브 콘텐츠

Python 기초, 이벤트 루프, 간단한 물리/충돌 처리, Unity 또는 웹 canvas 기초를 학습한다. 작은 게임을 끝까지 완성하는 경험이 중요하다.
