# 소프트웨어학부 트랙 안내

AI와 데이터 분석에 관심이 있는 학생은 Python, 수학/통계, 자료구조, 머신러닝 관련 학습 방향을 우선 탐색한다. 공식 트랙 안내의 빅데이터/머신러닝 방향과 연결해 상담한다.

웹과 서비스 개발에 관심이 있는 학생은 HTML, CSS, JavaScript, React, 데이터베이스, API 설계를 학습하면 좋다. 공식 트랙 안내의 웹/정보보호, 엔터프라이즈SW 방향과 함께 비교한다.

정보보호에 관심이 있는 학생은 네트워크, 운영체제, 웹 보안, 암호 기초를 함께 공부하는 것이 좋다. 트랙 선택은 학생의 적용 교과과정과 실제 개설 과목을 확인한 뒤 확정한다.
