# 인공지능학부 2025 교과과정 단계

인공지능학부 2025 교과과정은 기초 수학과 CS/SW 역량을 먼저 만들고, 그 위에 핵심 인공지능 과목과 융합 프로젝트를 쌓는 단계형 구조로 안내된다.

## Basic 단계

Basic 단계는 1, 2학년 학생이 고급 인공지능 학습에 들어가기 전 반드시 확인해야 하는 기초 구간이다. 주요 축은 선형대수, 미적분학, 수리통계학, 확률론 같은 수학 기초와 Python, 자료구조, 알고리즘, 데이터베이스 같은 CS/SW 기초다.

상담 기준은 다음과 같다.

- 인공지능학부 1학년이 과목 우선순위를 묻는 경우 수학 기초와 Python, 자료구조를 먼저 확인한다.
- 인공지능 모델 구현에 어려움을 겪는 학생에게는 모델 과목보다 자료구조, 알고리즘, 데이터베이스, 확률/통계 기초를 다시 연결한다.
- Basic 단계는 인공지능학부 학생이 반드시 수강해야 하는 기초 축으로 취급한다.

## Advance 단계

Advance 단계는 2, 3학년에서 CS/SW 기본 역량을 바탕으로 인공지능 개론, 기계학습 같은 핵심 인공지능 역량을 프로젝트 기반으로 확장하는 구간이다.

상담 기준은 다음과 같다.

- 인공지능 개론이나 기계학습을 듣기 전에는 Python 구현력, 자료구조, 알고리즘, 수학 기초가 충분한지 먼저 점검한다.
- 이론만 알고 실제 구현이 약한 학생에게는 프로젝트 기반 과목, 실습형 과제, 팀 프로젝트 경험을 함께 추천한다.
- 모델 성능 개선, 데이터 전처리, 평가 지표 같은 질문은 Advance 단계의 프로젝트 경험과 연결해 답변한다.

## Creative 단계

Creative 단계는 실전 문제해결 역량과 다른 전공 분야와의 융합을 강조한다. 생명공학, 물리, 기계, 예술 같은 분야와 인공지능을 연결해 미래 변화에 대응하는 융합형 인재를 목표로 한다.

상담 기준은 다음과 같다.

- AI와 다른 전공을 함께 고민하는 학생에게는 문제 도메인, 사용할 데이터, 팀 구성, 결과물 형태를 먼저 정리하게 한다.
- 포트폴리오 목적의 학생에게는 단순 모델 학습보다 실제 문제 정의, 데이터 확보, 평가, 서비스화까지 이어지는 프로젝트를 추천한다.
- 융합 프로젝트는 최신 기술 과목, 연구 프로젝트, 현장실습, 학부 연구실 활동과 함께 비교한다.
