# 발표자료 구성

발표영상은 20-25분을 목표로 합니다.

## 1. 주제 소개

- 북악 새내기 AI
- 국민대 신입생의 정보 탐색 문제
- 왜 RAG가 필요한가

## 2. 시스템 구조

- React UI
- FastAPI Python backend
- Supabase Auth/Postgres/pgvector
- Gemini API
- RAG 흐름

## 3. 기능 데모

1. 로그인
2. RAG 챗봇 질문
3. 출처 확인
4. 트랙/활동 추천
5. 자연어 일정 등록
6. D-day 확인
7. LLM 활용 기록 확인

## 4. 코드 설명

- FastAPI route 구조
- 추천 규칙 딕셔너리
- 조건문 분기
- chunk 반복 처리
- D-day 계산 함수
- LLM usage log 저장

## 5. LLM 활용과 협업 방식

- Codex로 PRD 분석, spec 작성, plan 작성, 문서 구조 설계
- Gemini API로 앱 기능 구현
- 팀원이 직접 검토하고 설명 가능한 코드로 수정

## 6. 한계와 개선 방향

- 자료 최신화
- 더 많은 학과 확장
- Google Calendar 연동
- RAG 평가 자동화
