# 팀원 데이터 요청서

이 문서는 KMU SW Navigator를 실제 서비스처럼 보이게 만들기 위해 팀원에게 추가로 받아야 하는 데이터를 정리한 것이다.

공식 사이트에서 교과과정, 졸업요건, 트랙, 공지, 장학, 행사, SW중심대학 프로그램 정보는 어느 정도 확보 가능하다. 부족한 부분은 공개 웹사이트에 정형 데이터로 없거나, 개인화 추천에 필요한 실제 사용자 맥락이다.

## 1. 동아리/소모임 데이터

공식 사이트에는 행사 공지는 있지만 동아리/소모임 목록과 추천 기준이 부족하다.

팀원에게 아래 형식으로 요청한다.

```txt
동아리명:
분야: AI / 웹 / 앱 / 게임 / 보안 / 알고리즘 / 데이터 / 창업 / 기타
한 줄 소개:
주요 활동:
모집 시기:
지원 방법:
추천 대상: 1학년 / 2학년 / 전공자 / 비전공자 / 초보 가능 등
활동 난이도:
공식 링크 또는 연락처:
비고:
```

## 2. 학생 개인화 샘플 데이터

개인화 추천이 실제처럼 보이려면 학생 프로필 예시가 필요하다. 실명이나 학번 전체는 받지 않는다.

```txt
학부:
입학년도:
학년:
관심 분야:
목표: 취업 / 대학원 / 창업 / 공모전 / 졸업요건 / 트랙 선택
이수한 과목:
현재 듣는 과목:
부족하다고 느끼는 점:
이번 학기 목표:
선호 활동: 동아리 / 대회 / 연구실 / 인턴 / 프로젝트 / 자격증 등
```

## 3. 실제 과제/시험/일정 데이터

Google Calendar 연동은 제외하고 내부 캘린더로 구현한다. 그래서 테스트용 실제 일정 데이터가 필요하다.

```txt
과목명:
일정 종류: 과제 / 시험 / 팀플 / 신청 / 행사
제목:
마감일:
중요도: 낮음 / 보통 / 높음
상태: 예정 / 진행중 / 완료
메모:
```

## 4. 진로/취업 추천 기준

공식 공지는 제목과 날짜 중심이라, 어떤 학생에게 어떤 활동을 추천할지 판단 기준이 부족하다.

```txt
목표 직무: 백엔드 / 프론트엔드 / AI / 데이터 / 보안 / 게임 / 임베디드 / 기획 / 창업
필요 역량:
추천 과목:
추천 활동:
추천 프로젝트:
추천 자격증 또는 대회:
관련 공지 또는 참고 링크:
```

## 5. 학교 생활 FAQ

공식 문서만 있으면 검색봇처럼 보이기 쉽다. 실제 학생이 자주 묻는 질문과 짧은 답변이 필요하다.

```txt
질문:
답변:
대상: 신입생 / 2학년 / 졸업예정자 / 전과생 / 다전공생 등
근거 링크:
주의사항:
```

## 6. RAG 과목/수강신청 데이터

과목 추천과 수강신청 상담 품질을 높이려면 공식 교과과정만으로 부족하다. 팀원에게 과목 후기, 수강신청 팁, 진로별 추천 과목 순서를 추가로 요청한다.

자세한 수집 기준은 `docs/architecture/rag-course-data-collection.md`를 따른다.

요청 항목:

```txt
과목명:
들은 학기:
난이도:
과제량:
팀플 여부:
도움 되는 사전 지식:
추천 대상:
수강신청 팁:
진로 연결: AI / 백엔드 / 보안 / 게임 / 데이터 / 기타
근거 유형: 공식 / 선배 경험 / 팀원 의견
확인일:
```

## 팀원에게 보낼 요청 문구

아래 문구를 그대로 보내면 된다.

```txt
KMU SW Navigator 개인화 추천 기능에 넣을 실제 데이터를 모으고 있습니다.

공식 사이트에서 교과과정, 졸업요건, 트랙, 공지, 장학, 행사 정보는 확보 가능하지만, 동아리/소모임 목록, 학생 개인화 샘플, 실제 과제·시험 일정, 진로별 추천 기준, 학교 생활 FAQ는 공개 사이트만으로 부족합니다.

가능한 범위에서 아래 항목을 채워서 보내주세요.

1. 알고 있는 동아리/소모임 정보
2. 익명 학생 프로필 예시
3. 실제 과제/시험/팀플/행사 일정 예시
4. 진로별 추천 과목/활동/프로젝트 기준
5. 신입생이나 재학생이 자주 묻는 질문과 답변

개인정보는 넣지 말고, 실명/학번 전체/전화번호 같은 민감정보는 제외해주세요.
```

## 우선순위

1. 동아리/소모임 목록
2. 익명 학생 프로필 3-5개
3. 실제 과제/시험/팀플 일정 10개 이상
4. 진로별 추천 기준
5. 과목 후기와 수강신청 팁
6. 학교 생활 FAQ

이 데이터가 들어오면 앱은 단순 공식문서 검색이 아니라 `학생 상황 + 공식 정보 + 내부 일정 + 활동 추천`을 함께 다루는 개인화 내비게이터로 보일 수 있다.
