# 데모 시나리오

발표영상은 20-25분을 목표로 합니다.

## 1. 프로젝트 소개

- 북악 새내기 AI 소개
- 신입생의 정보 탐색 문제 설명
- RAG가 필요한 이유 설명

## 2. 로그인과 메인 화면

- 데모 계정으로 로그인
- 주요 탭 소개: AI 챗봇, 추천, 일정, LLM 기록

## 3. RAG 챗봇

질문:

```txt
AI에 관심이 많은 신입생인데 어떤 트랙을 보면 좋을까?
```

보여줄 것:

- 검색된 문서 출처
- 답변 내용
- 근거 부족 시 제한적으로 답변하는 정책

## 4. 추천 기능

입력:

```txt
관심 분야: AI
목표: 취업
코딩 경험: 초급
학습 성향: 프로젝트
```

보여줄 것:

- 조건문/딕셔너리 기반 추천
- 추천 이유와 액션 리스트

## 5. 일정 관리

입력:

```txt
자료구조 과제 다음주 금요일 밤 11시 59분까지 제출해야 해
```

보여줄 것:

- Gemini JSON 추출 결과
- 저장 전 미리보기
- D-day 표시

## 6. LLM 활용 기록

- Gemini API 사용 기록
- Codex/Superpowers 기반 설계, 계획, 구현, 검증 기록
- 팀 협업 기록
