# 챗봇 근거 시각화 목업 디자인

## 배경

챗봇 답변이 텍스트만 있으면 “LLM이 그냥 생성한 답변”처럼 보일 수 있다. 이 프로젝트의 강점은 Mini LLM Wiki와 raw source를 활용한 RAG 흐름이므로, 챗봇 화면에서 답변 근거와 추천 흐름을 시각적으로 보여주는 목업이 필요하다.

## 목표

- 사용자가 질문을 입력하면 답변, 출처, 추천 로드맵, 근거 흐름을 한 화면에서 확인하는 구조를 제안한다.
- 발표에서 “입력 → 검색 근거 → 추천 판단 → 답변”을 설명할 수 있게 한다.
- 모바일에서도 채팅과 근거 패널을 세로로 읽을 수 있게 한다.

## 화면 구조

- 상단 메뉴바: 기존 `login.html`, `main.html` 목업과 동일한 일반 메뉴바 계열
- 상단 상태 영역: 현재 모드, 출처 표시, wiki-first 검색 상태
- 왼쪽/상단 주 영역: 채팅 메시지와 입력창
- 오른쪽/하단 근거 영역:
  - 답변 근거 그래프
  - 1학년 학습 로드맵
  - Mini Wiki와 raw source 연결 흐름
  - 추천 이유 체인

## 시각화 방식

MVP 목업은 외부 그래프 라이브러리를 사용하지 않고 HTML/CSS로 표현한다. 실제 React 구현 단계에서는 먼저 이 구조를 custom SVG/CSS로 옮기고, 드래그/줌이 필요해지면 React Flow를 검토한다.

## 데이터 모델 구성

```json
{
  "question": "AI 트랙에 관심 있는 1학년은 뭘 먼저 준비하면 돼?",
  "answer": "...",
  "sources": [
    {"type": "wiki", "title": "커리큘럼 로드맵"},
    {"type": "raw", "title": "소프트웨어학부 트랙 안내"}
  ],
  "roadmap": [
    {"term": "1학년 1학기", "items": ["Python 기초", "수학 기초"]},
    {"term": "1학년 2학기", "items": ["데이터 분석", "프로젝트 경험"]},
    {"term": "2학년 이후", "items": ["AI/데이터 트랙 탐색"]}
  ],
  "reasonChain": ["AI 관심", "코딩 경험 조금 있음", "Python 실습", "프로젝트 활동"]
}
```
